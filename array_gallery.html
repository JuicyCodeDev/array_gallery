.filter()

The filter method runs a provided callback function on every element of the array. It returns a new array with all elements that pass the call back - which means the return value of the callback is equal to "true".


Syntax

let newArr = arr.filter(function callback(currentValue){
//callback function body
});


Example

let ages = [12,24,34,18,16,14,22,25];

function isFullAge(age){
(age>18) ? true : false;
}

let fullAges = ages.filter(isFullAge); console.log(fullAges) // [24,34,18,22,25]

.sort()

The sort() method sorts the items of an array. The default sort order is ascending, built upon converting the elements into strings, then comparing their sequences of UTF-16 code units values. Because of this, the sort() method will produce an incorrect result when sorting numbers. You can fix this by providing a "compare function”. When the sort() method compares two values, it sends the values to the compare function, and sorts the values according to the returned value. Therefore the compare function should return a negative, zero, or positive value.

1. If compare (a,b) is less than zero, the sort( ) method sorts a to a lower index than b. In other words, a will come first.

2. If compare (a,b) is greater than zero, the sort( ) method sort b to a lower index than a, i.e., b will come first.

3. If compare (a,b) returns zero, the sort ( ) method considers a equals b and leaves their positions unchanged.


Syntax

let newArr =
arr.sort(compareFunction(a, b){
//compare function body
});


Example

let nums = [40, 100, 1, 5, 25, 10];

nums.sort(function(a, b) {
return a - b)
});
// nums is now [1, 5, 10, 25, 40, 100]

.map()

The .map() method run a provided callback function on every element of the array. It returns the results that are returned by the callback each time in a new array.


Syntax

let newArr = arr.map(function callback(currentValue){
//callback function body
});


Example

let numbers = [1,4,9,16,25,36,49,64];

let roots = numbersmap(function(num) {
return Math.sqrt(num)
})
// roots is now [1, 2, 3]
// numbers is still [1, 4, 9]

.reduce()

The .reduce() method executes a reducer function (that you provide) on each element of the array, resulting in a single output value.


Syntax

let newArr = arr.reduce(function callback(accumulator, currentValue){
//reducer function body
},initialValue);


Example

let sum = [0, 1, 2, 3]
.reduce(function(accumulator, currentValue) { return accumulator + currentValue }, 0)
// sum is 6

.flat()

The flat() method creates a new array with all sub-array elements concatenated into it recursively up to the specified depth. The depth level is specifying how deep a nested array structure should be flattened. It defaults to 1.

Syntax

myArray.flat(depth);


Example

const myArr = [ 1,2, [3, 4, [5, 6]]];
myArr.flat();
// [1, 2, 3, 4, [5, 6]]

const myArr = [ 1,2, [3, 4, [5, 6]]];
myArr.flat(2);
// [1, 2, 3, 4, 5, 6]
